# Задание 4.1 "Контейнер Set"
![CI](https://github.com/itsOleggio/AdvancedJS_T4_1/actions/workflows/ci.yml/badge.svg)

## Описание 


Программа для работы с добавлением игроков в команду и хранением их в виде коллекции Set. Добавляя игроков в команду, программа проверяет, что такого игрока еще нет в команде.

## Тесты

Используем библиотеку Jest.
Тестом покрыта основная функция программы 
- `add(character)` - добавление игрока в команду
- `addAll(...character)` - добавление игроков в команду
- `toArray()` - получение всех игроков в команде в виде массива


Запуск тестов: `npx test`. 

## Код программы

Программа находится в папке `src`.
Тесты ёт в папке `/src/ts/__tests__/`.

## Запуск программы

Сборка программы: `npm build`
Запуск программы: `npm start`.
